# 🖧 CCNA Cisco Labs

Hands-on Cisco networking labs and command references based on real CCNA exam topics. Each folder contains a fully worked lab with tasks, step-by-step configuration commands, and verification steps.

All labs can be practised using **GNS3**, **Cisco Packet Tracer**, or **EVE-NG**.

---

## 📁 Lab Structure

| Folder | Topic | Difficulty |
|---|---|---|
| [`/vlans-and-trunks`](./vlans-and-trunks/) | VLAN creation, trunking, DTP, ISL vs 802.1Q | 🟢 Beginner |
| [`/vtp`](./vtp/) | VLAN Trunking Protocol — Server, Client, Transparent | 🟢 Beginner |
| [`/etherchannel`](./etherchannel/) | Link aggregation using LACP and PAgP | 🟡 Intermediate |
| [`/inter-vlan-routing`](./inter-vlan-routing/) | IVR (L3 Switch) and Router-on-a-Stick (ROAS) | 🟡 Intermediate |
| [`/spanning-tree`](./spanning-tree/) | STP, Root Bridge election, port roles | 🟡 Intermediate |
| [`/routing`](./routing/) | Static, Default, RIP, EIGRP, OSPF | 🔴 Advanced |
| [`/acl`](./acl/) | Standard and Extended Access Control Lists | 🔴 Advanced |
| [`/nat`](./nat/) | NAT and PAT configuration | 🔴 Advanced |
| [`/dhcp-dns`](./dhcp-dns/) | DHCP server setup and DNS configuration | 🟡 Intermediate |
| [`/security`](./security/) | L2 Security, Password hardening, SSH | 🔴 Advanced |
| [`/concepts`](./concepts/) | Theory notes — VLANs, STP, Routing protocols | 📘 Reference |
